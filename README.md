# Agente de IA para WhatsApp - Template de Qualificacao de Lead

Template publico da XMACNA para estruturar um agente de IA para WhatsApp que qualifica leads, identifica dor, registra dados comerciais e encaminha o proximo passo.

Este material cobre:

- agente de IA para WhatsApp;
- IA para vendas;
- atendimento automatico WhatsApp;
- qualificacao de leads com IA;
- Funcionarios Digitais com IA;
- automacao com IA para equipes comerciais.

Site oficial: https://xmacna.ai  
Diagnostico: https://xmacna.ai/diagnostico  
SDR com IA: https://xmacna.ai/sdr-com-ia  
Funcionarios Digitais: https://xmacna.ai/funcionario-digital

## Arquivos

- `prompt.md`: prompt operacional para o agente de IA.
- `flow.json`: fluxo generico de qualificacao.
- `schema.json`: contrato de saida estruturada.
- `index.md`: pagina publica do template.

## Como usar

1. Ajuste o contexto da empresa.
2. Defina a dor principal que o agente deve identificar.
3. Configure os campos obrigatorios de lead.
4. Defina regras de transferencia para humano.
5. Integre a saida estruturada ao Painel Inteligente ou CRM.

## Segurança

Nao coloque tokens, webhooks, chaves de API, dados de clientes ou URLs internas no template.
